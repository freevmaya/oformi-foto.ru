Application usage guide

Description of the application interface

The application interface for photo design consists of the main five blocks

  • The largest block in the application: the working field, shows the current edition of the design of the photo
  • On the right, the selected photo with the controls:
    • Positioning the selected area in the photo
    • Mirror photo rotation horizontally
    • Mirror vertical rotation
    • Correction of brightness, contrast and color photos
  • The first lower block, a tape of patterns in two rows with buttons of horizontal scrolling. To select the desired template, click on it
  • The second lower block, a ribbon of your photos grouped by album. (only for authorized users)
  • Upper right block, application plug-ins buttons. To date, this is:
    • "As for a cell phone" - the button opens a dialog for selecting the type of phone, allows you to save the created postcard for use as a background on your cell phone
    • "Add music to the postcard" - adds sound to the postcard, this feature is only available for users authorized in "My World"
    • "Upload photo" - loads a photo from the computer for registration. This feature is available and not for authorized users. The downloaded photo is not saved and can be used only in the current session.

Toolbar

To be under the working field, includes several buttons:

  • "Full size" - hides all the controls and deploys the current layout to the entire workspace of the application, if the created layout is larger than the working area, it will be inscribed in the size of the available area.
  • "Incoming cards" - only for authorized users. Opens a dialog with a list of incoming postcards that were sent by your friends from the application.
  • "Outgoing cards" means postcards you have sent that have not yet been viewed by the addressee.
  • "Add effect" - adds to the postcard a visual effect, an animated decoration of your postcard. Effects are only used for postcards sent from the application.
  • "Save Card" - saves the created postcard to the user's computer in a JPEG graphic file format. No effects and music.
  • "Your balance" is the balance of the user in the application. Replenishment is possible only from the application on the site "My World"
