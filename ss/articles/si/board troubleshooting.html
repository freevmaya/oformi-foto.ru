Разрешение проблем

Появилось сообщение о том что Flash player незагрузился

  1. См. действие 1.

Приложение не работает

  1. Не загружается Flash Player, видно надпись "Приложение загружается", дальше ни чего не происходит. См. действие 1.
  2. Flash Player загрузился
    • 2.1. Видно индикатор загрузки, дальше ни чего не происходит, см. действие 5.
    • 2.2. Появляется сообщение "Приложение не может соединиться с сайтом одноклассники.ру!", см. действие 5.
    • 2.3. Происходит что то другое, см. действие 4.
  3. Происходит что то другое, см. действие 4.

Действия

  1. Проверить установку и включение Flash Player
    • 1.1 Mozzila Firefox
      В меню "Инструменты"->"Дополнения"->"Плагины". В списке найти "Shockwave Flash"
      если нет то см. действие 2
      если есть то см. действие 3.
    • 1.2 Opera
      В левом верхнем углу кликнуть левой кнопкой на иконке "Opera", в меню выбрать "Инструменты"->"Дополнительно"->"Плагины".
      Проверить включение плагинов, если нет то включить
      В списке найти Shockwave Flash
      если нет тогда см. действие 2
      если есть то см. действие 3.
    • 1.2 Chrome
      Зайдите в настройки (в меню справа, сверху выберите: "Настройки"). Найдите раздел "Личные данные", возможно потребуется раскрыть список ссылкой "Показать дополнительные настройки". Далее выберите "Настройки контента", найдите раздел "Flash". Выберите "Разрешить Flash на сайтах", либо добавьте наш сайт в исключения.
      То же самое можно сделать если вы кликните слева от адресной строки сайта.

      Для более старых версий барузера, в строке адреса введите chrome://plugins/, нажмите ввод
      В списке найдите "Adobe Flash Player"
      если есть, то см. действие 3.
      если нет тогда см. действие 2.
    • 1.3 Internet Explorer
      в меню "Сервис" выбрать "Надстройки"
      в списке найти "Shockwave Flash Object"
      если есть, то см. действие 3.
      если нет тогда см. действие 2.
  2. Установить Flash Player с сайта adobe.com
  3. Проверка плагина
    • 3.1 Проверить включение и разрешение, если выключен то включить, если не разрешено тогда разрешить. Перезагрузить браузер.
    • 3.2 Версия плагина должна быть не ниже 11, если установленная версия ниже 11 то см. действие 2.
  4. Пожалуйста опишите подробно и отправьте письмо на e-mail
    • 4.1 Суть проблемы, адрес приложения которое вы открываете, сообщения и ошибки которые появляются на экране
    • 4.2 Сообщите, если есть, номер внизу приложения под названием "Ваш ID"
    • 4.2 Название и версию операционной системы установленной на вашем устройстве
    • 4.3 Название и версию браузера в котором вы открываете приложение
  5. Возможно в связи с некорректными настройками браузера приложение не загружается
    • 5.1 Попробуйте открыть приложение в другом браузере
    • 5.2 Установите другой браузер и попробуйте открыть приложение (см. ниже ссылки на популярные брайзеры)
      • Google Chrome
      • Mozilla Forefox
      • Opera
      • Internet Explorer
      • Safari

Термины

  1. Браузер - программа для просмотра веб-сайтов
  2. Операционная система - комплекс управляющих и обрабатывающих программ обеспечивающих взаимодействие пользователя и компьютера, например Windows, Mac OS, Unix, Android и .т.п
  3. Плагин - независимый программный модуль, динамически подключаемый к основной программе и предназначенный для расширения и/или использования её возможностей.
